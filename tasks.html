DOM_JS. Считываем информацию из DOM

Ниже представлен макет вымышленного паспорта. Задача — используя Инструменты разработчика, получить данные из DOM.

г. Самара

Инструкция

1. В вашем браузере откройте "Инструменты разработчика" (обычно клавиша F12 или Ctrl+Shift+I).

2. Перейдите на вкладку "Консоль" (Console).

3. Введите в консоли следующие команды (по одной) и нажмите Enter, чтобы получить данные:

// Получить Фамилию:

document.getElementById('read-lastname').innerText

// Получить Имя:

document.getElementById('read-firstname').innerText

// Получить Год рождения:

document.getElementById('read-birthyear').innerText
